## Scripts

### Runs the app in the development mode.

```bash
npm run dev
```

### Builds the app for production.

```bash
npm run build
```

### Runs the app in the production mode.

```bash
npm run start
```

### Lint the code.

```bash
npm run lint
```

### Lint, fix, and format the code.

```bash
npm run lint:fix:format
```

### Format the code.

```bash
npm run format
```

### Test the code.

```bash
npm run test
```

### Test the code in CI.

```bash
npm run test:ci
```
